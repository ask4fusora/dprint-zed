# dprint extension for Zed

This extension adds support for [dprint](https://github.com/dprint/dprint) in [Zed](https://zed.dev/).

Languages currently supported:

- **JavaScript**
- **TypeScript**
- **JSX**
- **TSX**
- **JSON**
- **JSONC**
- **Vue.js**
- **Astro**
- **Svelte**
- **CSS**
- **GraphQL**
- **HTML**

## Installation

Requires Zed >= **v0.131.0**.

This extension is available in the extensions view inside the Zed editor. Open `zed: extensions` and search for _Dprint_.

## Documentation

Head to the [official documentation](https://biomejs.dev/reference/zed) to learn how to set up the extension, and customize it.
